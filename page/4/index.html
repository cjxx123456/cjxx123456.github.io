月亮 by hxs

发表于 2024-08-24 更新于 2024-09-19

“举头望明月，低头思故乡。”“但愿人长久，千里共娟。”古人对月亮的喜爱之情可谓着迷，它在那时是那样神秘。古时人们认为月亮上有一座广寒宫，嫦娥就在里面居住。

到了现在，人们对月球的探索还没有结束。以前，月球一直以正面面向我们，我们无法看见它的背面。有人就说月球背面上有外星人。前不久，中国的“嫦娥六号”飞船为了解开这个疑问，飞到了月球的背面，还带回来了月球“土特产”，也就是月球土壤。

我见过许多的月亮，有被云遮住的月亮，有半残的月亮，有黄得发红的月亮。这些月亮总是高挂在那乌黑的天空，像一盏灯点亮了这乌黑的大地。

人们望月亮会想起什么呢？

人们望着同一轮明月，难免会想起远在他乡的亲人。月亮圆圆的，象征着团圆，让人想起与家人一起团聚的回忆。当遥望那洁白无暇的月亮时，远方的家人也许也看着月亮，好像有了心灵感应一般能够体会到双方的思念。对于远在他乡的游子来说，与家人团聚是多好的一件事。

我多想在洁白的月光照亮大地时，来到河边，观看月亮，河上会映出另一个月亮，想要把它捞起，却只能捞上一些河水。

每当想到黑夜之后，月亮便会离开我，我就有一点感伤。因此每到傍晚，我都会到桥上迎接月亮，等待她再次照亮黑夜。

月亮 by ymx

发表于 2024-08-24 更新于 2024-09-04

月亮，这个古老而神秘的天体，承载着无尽的情感。举头望明月。晚上的时候，看见那温馨而美丽的月亮，心中就有了很多对月亮的想象。

在宁静的夜晚，月亮悄然升起，它那洁白的光芒洒满了大地，如同一抹温柔的微笑。她那圆圆的脸庞有些嫣红，也许是暗暗的粉红。月亮懒懒地向上爬，一纵一纵的，看起来很累。费了好大的劲，月亮才冲出了地平线，挂在了半空中。这时，她脱去了红装，换上了金黄色的纱巾，用白玉般的笑脸来迎接我们这些芸芸众生。

我非常喜欢看天空，特别是晚上的天空。每次一吃完饭，我都会立与跑出去，抬头望着天上的月亮和星星，星星一闪一闪的，月亮的光照亮了一整个小院，犹如一盏大灯。有些边角角也一样照亮了。月亮有几种不同的颜色:红色，黄色、白色等。月亮或许是五颜六色的，或许它只有一种颜色“白色”。月亮有时的大小也都不一样。在中秋节的时候，它是最大而又圆润的，不知月亮在什么时候是弯弯的，像一把镰刀。

月亮也承载着我对祖祖的情感。小时候很调皮，把我祖祖的眼睛给打了，那时还不懂事，现在长大了，就莫名的有种后悔，愧疚感。所以我想把月亮看成我的祖祖，这样每天晚上我都能看见他了。有时看月亮，就会猜想这是不是祖祖的心情呢？

月亮照亮了我的心灵，也照亮了我的生活。“今人不见古时月，今月曾经照古人。”

我的一天 by wyx

发表于 2024-08-23 更新于 2025-07-06

一天，二十四小时，每分每秒都在指尖逝去，想捧，却也捧不住。

我的一天，宁静致远，淡然处之…

周六，我回了老家。那天我比以往起得早很多，差不多天还没亮就起床了。吃完早饭，来到屋后的山脚，山上的风景很美，山清水秀，美得犹如一幅画。

早晨的雾气很大，一眼望去竟然看不到山顶，只是朦胧一片，再露出点点草色，像是一个穿了纱衣的仙女侧卧在我们面前。山上的绿色由深到浅的排列着，还夹杂着冬日未脱去的枯草。时不时拂来一阵醉人的风。

奶奶说今天要去种小白菜，那块地之前是种菜花的，后面丰收了就没种过什么了。看起来光秃秃的。

途中，看到了奶奶种的核桃树。她跟我讲，之前山上下了一次很大的雨，种的那些梅子树都被吹走了，唯独这株核桃树在大风中侥幸留了下来。它或许是靠自己的毅力才没被吹走。现在它长得很高，像一个笔直的少年伫立在那里。我很惊讶，它不惧风雨还安静地站在那里，仿佛一切与它无关。

来到菜地，草已经长得很茂盛，不得不先锄掉。接着，拿起锄头挖一个洞，丢一颗种子进去，用土盖上，不能太紧，不然长不起来。就是这样重复，直到种满一片。这样简单的动作，却需要极大的耐心，没过一会我就累得不行。奶奶告诉我不能半途而废，我也明白这个道理，我们一起在这片土地里种满希望。

回到家，我疲倦地躺在床上，回想起顽强的核桃树，等待收获的小白菜，这些构成了我那平淡而又安心的一天。

我的一天 by ljyq

发表于 2024-08-23

早上，阳光照进房间，给人一种温暖又舒适的感觉。空气十分清新，我伸了个懒腰，准备去上学。

去学习的路上，马路上的积水倒映出天空的模样，十分美丽。小鸟也在忙碌地搭着自己的小窝。空气中还带有一点清单的花香。路上有许多忙碌的人，有的要上班，有的要去山上摘茶拿去卖钱。在路上看到冰棍就顺手买了一根，背着书包，啃着冰棍去上学。感觉我来的路上，都是夏天的样子。

到学校后，教室已经坐满了人。上课的时候，我看到窗口有一朵山茶花。她的样子十分美丽，像一位高贵的妇人，穿着她红色的礼服，在阳光下尽情舒展。窗外还有小鸟在叽叽喳喳地演唱，十分的好听。

时间很快来到了下午，同学们都前往食堂吃饭。夕阳洒在操场上，将操场变成红黄色的。晚风轻轻吹动，凉丝丝的风里还带着花瓣，一起吹到我身前。看到这风景，我的第一反应就是小说里才会有的景象。那样子，美得无法用词来形容。

吃完晚饭后，就放学回家了。天空变的很暗，月亮也慢慢探出了头。一个人走在大桥上，没有行人，没有车辆，只有月亮与我相伴。它就像夜灯一样，为我照亮前方的路。大桥上的风吹的我凉丝丝的，溪水流动的声音很好听，路边的虫儿吱吱叫，就像大自然的野生歌手一样。

一天的疲倦都发泄到了床上，躺下后就像躺在了时间里。

月亮

发表于 2024-06-20 更新于 2024-09-04

一缕轻柔的月光透过窗子，洒在了窗台上，像镀了银一样。

我走出房间赏月。月亮弯弯的，古人说，那是被天狗吃了，可我不信，毕竟天上没有狗。低头一看，地上有一些杂草，上面还有一些露水，让小草弯下了腰。露水在月光的照射下，十分美丽，晶莹剔透。这让我想起小学时背过的古诗：“可怜九月三夜，露似珍珠月似亭”。这样美丽的景色，怎么不会让人们坐下来欣赏呢？

我走到池塘边。池塘很小，却足以容下一整个月亮。池塘是水泥做的，鱼儿也是普通的鱼。可是在月光的照射下，好像在一个由银子制成的池子里有一群“银鱼”要抓到月亮。一阵微风吹过，几片竹叶被吹进了池塘中。鱼、竹叶、倒映在水中的月亮，成了一幅美丽的“风景画”。这幅画是独一无二的，是世界上最美丽的“画”。

月亮不只是有最美的景色，还是思念的化身。

“举头望明月，低头思故乡”，这一刻，即使人们相隔千里，但同时抬头看着月亮，双方就好像就在面前。我们的知音不可能一直在身边。离别是我们人生中必须经历的一环。这不由地让我想起苏轼的《水调歌头》：人有悲欢离合，月有阴晴圆缺，此事古难全。但愿人长久，千里共婵娟。

每当望向着皎洁的月亮时，我都会想起去世的奶奶。如果那天不是星期五，而且周六，周天、那我的遗憾就不会是没见奶奶最后一面。奶奶对我很好，小时候爸爸打我时，奶奶会护着我。从街上回来，她会给我买东西。也许奶奶也会在另一个世界看着我。

“明月几时有，把酒问青天。”今晚的月亮多美啊！

月亮 by lmy

发表于 2024-06-17 更新于 2024-09-04

又是一个寂静的夜晚，我漫步在小区的花园里。抬眼望去，那一轮皎洁的明月高悬天际。月光如水，洒下清冷的光辉，给整个世界蒙上了一层银纱。

我静静地坐在那里，沐浴在月光之中，脑海里思绪万千，心中涌起一股浓浓的思乡之情。我抬头望向那一轮明月，朦胧的月色中，我好像看见了以前的时光。曾经的那些场景，清晰地浮现在眼前，是那么的熟悉，那么的温暖，像是昨天才发生的。

我想起来了远方的亲人。一起度过的欢乐时刻，如电影般在我的脑海中不断回放：我和弟弟在月关下嬉戏玩耍，分享着彼此的梦想，笑声回荡在寂静夜空。如今我们天各一方，只有通过这月光传递彼此的思念。

月光也让我想起曾经的挚友。我们三人一起上学，一起写作业，一起谈天说地，是那么的美好，那么的难忘。记得我们捡过一只蝴蝶，不敢带入教室，所以将它放在一颗小树旁，每节课下课都会去看它，它也在那里等着我们。后来，蝴蝶还是死了。我们将它埋好后，还给它立了一个碑，约定每天都要来看它。这是独属于我们三个的约定。我们还曾在月光下承诺，要一直在一起。可随着时间的推移，我们渐行渐远。此时此刻，他们是否也在看着这一轮明月，是否记起了我们的时光与誓言？

生活，像月亮，总会缺点，少点。但月亮一直都在，生活也还在继续，不断产生美好回忆。这些回忆就像风一样，朝我呼啸而来，却又匆匆离我而去。我尝试去捉住风，可我捉不住它，也无法停留在那些美好的时光。但我并没有因此而忧愁，反而感到庆幸。庆幸风经过了我，而我经过了那些时光。

我的一天 by zhy

发表于 2024-06-17 更新于 2024-06-20

每天早上，一束光照亮昏暗的房间，我便从蒙眬的梦中醒来，穿上衣服，走出房间，吃了饭后便出门上学。我的一天就这样开始了。

在我上学的路上。花儿在笑，小鸟在歌唱，有许多的花朵，在彼此推着挤着，好不活泼热闹。

我的家乡很美丽，有香甜可口的梅子，有巴掌一样大的梧桐树叶，有比天还高的大树……尽管有这么好吃的东西，有这么好看的风景，都比不过学校同学，老师的热情。

不知不觉间到了学校。远远远望去，操场上的蔷薇，还是一如平常的茂盛，紫色的大幅条上泛着点点银光。小朋友们也在广阔的前操场上无忧无虑地玩耍。看着他们，我也无比开心。

走进教室，看见了很有年代感的讲台和课桌。我走向自己的位置，周围同学们在玩烟卡。我快速放下书包，愉快地加入他们。

上课时，我认真听讲，没过多久就下课。课间我也没闲着，在那背书。

就这样放学了。我很快地跑出校门，不为别的，就为了吃一口学校门前的冰糕。

在回家的路上，我看见了刚下班的工人。他们一脸忧心忡忡，似乎并不开心。

我希望全天下的人都可以像我的一天一样，开开心心地生活。
